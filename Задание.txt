Необходимо реализовать Android – приложение, позволяющее вести список покупок.
Язык программирования Java или Kotlin

Приложение должно состоять из 3-х экранов:
    · cписок покупок;
    · экран добавления нового элемента;
    · экран с детальной информации.

Экран «Список покупок» должен содержать список уже добавленных элементов (RecyclerView) и кнопку 
перехода на экран создания нового элемента. Каждый элемент содержит название и фото товара. Нажатие 
на элемент открывает экран с детальной информацией.

Экран детальной информации отображает название, описание и фото товара. Экран должен поддерживать 
листание в стороны для перехода к следующему и предыдущему элементу списка товаров (ViewPager + Fragment).

Экран добавления нового элемента должен позволять ввести название товара, описание, и выбрать фото 
из галереи (возможен вариант с получением фото с камеры). Сохранение нового элемента должно происходить 
при закрытии экрана (можно предусмотреть отдельную кнопку для выхода без сохранения, но не обязательно).
Хранение списка покупок должно осуществляться в SQLite.

Выбор дополнительных инструментов на усмотрение разработчика.
